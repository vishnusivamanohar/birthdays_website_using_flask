A webpage that tells about Birthday website

This website is designed to help users keep track of birthdays of their friends and family. Users can add and delete birthday entries, and it shows the current day birthdays in the index. The goal of this website is to ensure that no important birthday is forgotten!

It contains three pages :

  • Home page: where users can see the list of birthdays and contain other options.
  • Input page: which can take input of birthdays.
  • Login page: Every user must sign up or login to create an account.

HOME PAGE:

This page shows today's birthdays and allows users to perform options by clicking the buttons below:

          1.  birthday input
          2.  view birthdays
          3.  logout

Birthday Input Page:

This page allows the user to add new birthday entries.
The user can input the name and date of birth of their friends or family members. After submitting the form, the new birthday will be added to the database.
